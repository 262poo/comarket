# Sesión 9: Interfaz gráfica de usuario

## Objetivo

Construir la primera interfaz gráfica del sistema, organizando vistas, controladores, componentes visuales y eventos de usuario.

## Preparación en IntelliJ

- Agregar dependencias o configuración de JavaFX.
- Crear paquetes `app.controller` y `app.view`.

## Contenidos

- Estructura base de una aplicación de escritorio con interfaz gráfica.
- FXML y separación vista-controlador.
- Componentes comunes de formularios.
- Eventos y navegación básica.

## Práctica guiada

- Crear una ventana de registro de productos.
- Enlazar controles con su controlador.
- Capturar eventos de botones y campos.

## Reto de sesión

Agregar una segunda vista o una tabla para listar registros.

## Entregable mínimo

Interfaz gráfica funcional que permita registrar datos y responder a eventos del usuario.

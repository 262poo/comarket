# comarket-cli

Proyecto de consola para la Unidad 1.

Aquí se trabaja el producto base del curso: clases del dominio, encapsulamiento, relaciones entre objetos, herencia o interfaces cuando correspondan, servicios, colecciones y CRUD en memoria desde consola.

La aplicación de escritorio JavaFX para Unidad 2 y Unidad 3 se trabaja en `../comarket-desk`.

# Programacion Orientada a Objetos

Recursos del curso de POO 2026-2 de la Universidad Peruana Union.

Este repositorio acompana la construccion progresiva de un sistema orientado a objetos. La ruta inicia con una aplicacion de consola, avanza hacia una aplicacion de escritorio con arquitectura por capas y persistencia relacional, y culmina con un sistema integrado para un proceso transaccional de negocio.

## Enlaces

- Libro digital: https://262poo.github.io/comarket/
- Repositorio: https://github.com/262poo/comarket
- Silabo 2026-2: https://262poo.github.io/comarket/silabo_poo_2026_2/

## Organizacion

```text
comarket/
  comarket-cli/      Producto U1: consola, POO y gestion de datos en memoria
  comarket-desk/     Producto U2-U3: escritorio, capas, persistencia y flujo transaccional
  docs/              Libro digital, sesiones, talleres, evidencias y silabos
```

## Productos del curso

- Producto U1: aplicacion de consola orientada a objetos con gestion de datos en memoria.
- Producto U2: aplicacion de escritorio por capas con interfaz grafica y persistencia relacional.
- Producto U3: sistema orientado a objetos integrado para un proceso transaccional de negocio.

## Uso sugerido

1. Revisar el libro digital desde `docs/index.md` o desde la publicacion web.
2. Trabajar las sesiones S01 a S16 en orden.
3. Usar `comarket-cli` para el producto de consola de la Unidad 1.
4. Usar `comarket-desk` para el producto de escritorio de las Unidades 2 y 3.
5. Registrar evidencias, aplicar observaciones del docente y preparar la sustentacion tecnica.
